# An e-commerce based website for renting electronic goods.
### Technologies used:
- Php
- Html
- CSS
- Jquery
- Bootstrap
### Features
- Fully responsive web design.
- Cart system to add products.
- Email system to send confirmation of orders.
- Admin panel to add,edit products.
